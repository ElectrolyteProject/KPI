命名规则：
ep-序号 SMILES化学式

产生规则：
1. 初始为C、O、F三个结点（实际是CH4、H2O、HF）；
2. 根据以下六种结构组合添加新结点；
    -C -O -F =C =O ≡C
3. 根据广度优先搜索组合添加单键、双键与三键，具体为借助队列为辅助队列，
将已添加新结点的图作为根结点，入队，出队后添加边，查找已有的图判断同构，
未生成当前结点数当前边数的图或与已存在的图不同构，入队，循环。

约束规则：
1. 氧不与氧、氟成键（不产生过氧键）；
2. 相邻的两个键不能同时为双键；
3. 单个分子的碳原子总数小于等于8，氧原子总数小于等于4；
4. 生成的图不能是非平面图（即与K5，K3,3子图同构的图）；
5. 数据库中任意两个图不同构；
6. 不产生与特定图子图同构的图（主要避免环张力过大）。
